##### **Niji**
> Turn your network packets into a command-line rainbow.
</div>

## Installation

```shell
git clone https://github.com/picatz/niji.git
cd niji
make
make install
```

## Usage

Just run `niji` to start it.
```
$ niji
```

Specify a network interface to listen on.
```
$ niji eth0
```
